Class: MarkerLabelLib

MarkerLabelLib

Used to represent and manage labels for the markers in Google Maps JavaScript API. This library has a set of common alphabets you can use, stored in UTF-8 format, so it is advised that if you use this script to have proper UTF-8 support in your code.

Constructor

new MarkerLabelLib(alphabet, recycleSymbols)

Creates a MarkerLabelLib object with the given alphabet and a Boolean to decide if the library should reuse deleted labels or not. The user can either provide one the alphabets present in this library, or something else, as long as it is an array of charaters.
Parameters:
Name Type Description
alphabet Array Array containing the alphabet to be used.
recycleSymbols Boolean Defines if the library should reuse deleted labels or not.

nextSymbol() → {String}

Returns the next symbol available in the collection. If symbol recycling is enabled, it will priotize the last symbol put in the recycling bin instead, until all deleted symbols are reused. Once it empties the recycle bin, it starts using the collection again where it left it the last time.

recycleSymbol(symbol)

Frees the given symbol (aka label), putting it in the reusable list of symbols. If you opted by not recycling the symbols you should not need to use this method, but if you use it be aware that the library will always track the symbols you free in case later on you change your mind and decide to enable the symbol recycling feature.
Parameters:
Name Type Description
symbol Char The symbol to be recycled.
