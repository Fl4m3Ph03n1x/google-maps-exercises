Class: MarkerTracker

MarkerTracker

Used to manage markers and their properties in Google Maps JavaScript API. This library tracks markers and any additional information you may want to associate with them, such as infoWindows or others. It also allows the user to associate listeners, easing their use.

Returns the last TrackerId used for a marker. TrackerIds are unique and never repeat, even when the markers using them are removed from the system. TrackerIds are not recyclable and thus can also be used as marker global identifiers.

Usage of internal from PrivateNameSpace

   get getX(){
       return super.internal(this).x;
