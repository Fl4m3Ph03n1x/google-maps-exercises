createMarker(aPosition, aTitle, aLabel, isDraggable, anAnimation) → {Object}

Creates and returns a marker with the given parameters. Depends on the global variable map.
Parameters:
Name Type Description
aPosition Location The Position object (containing a lat and lng properties) with the coordinates of the marker.
aTitle String The title of the marker.
aLabel Char OPTIONAL, The label of the marker. Markers can only have labels with 1 (one) character. If it is not set, defaults to to empty ''.
isDraggable Boolean OPTIONAL, If the marker is draggable or not. If not set, defaults to false.
anAnimation Object OPTIONAL, The animation of the marker when it is added to the map. If not set, defaults to google.maps.Animation.DROP.
