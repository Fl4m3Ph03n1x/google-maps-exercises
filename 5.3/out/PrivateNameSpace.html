Class: PrivateNameSpace

PrivateNameSpace

Class used to simulate private variables in objects in JavaScript. This class must be extended in order to be used. Classes extending this will be able to have private variables and will be tracked.

Example

Usage of internal from PrivateNameSpace

 class Point extends PrivateNameSpace{
    
   constructor(x, y){
       super();     
       super.internal(this).x = x;
       super.internal(this).y = y;
   }
  
   //Repeat for y
   get getX(){
       return super.internal(this).x; 
   }

   //Repeat for y
   set setX(newX){
       super.internal(this).x = newX; 
   }
}
